# 트리노드 AX 포트폴리오 계획

## 목표

트리노드 AX 인턴 지원용 포트폴리오로, 포코팡 계열의 한 붓 그리기 Line-Linker 퍼즐을 Unity 2D로 모작한다. 단순 클론이 아니라, AI 기반 게임 제작 역량을 보여주기 위해 "게임 기획자 AI 에이전트"가 레벨 목표, 보드 난이도, 플레이 지표를 분석하고 제안하는 구조를 함께 만든다.

## 채용공고 연결점

- 공고 키워드: Unity, AI, 게임AI 개발, 바이브코딩, 캐주얼 모바일 게임.
- 회사 대표작: 포코팡, 포코포코, 포코팡타운.
- 포트폴리오 메시지: "트리노드식 캐주얼 퍼즐 감각을 이해하고, AI/CLI 워크플로우로 기획-구현-튜닝 루프를 빠르게 구축할 수 있다."

## 게임 콘셉트

가칭: Poko Line Lab

- 장르: 모바일 캐주얼 Line-Linker 퍼즐.
- 핵심 조작: 같은 타입의 타일을 드래그로 이어서 3개 이상 연결하면 제거.
- 핵심 재미: 짧은 판단, 연쇄 제거, 목표 달성, 캐릭터/몬스터 공격 연출.
- 포트폴리오 차별점: AI 기획자 에이전트가 보드 상태와 플레이 로그를 읽고 다음 레벨 목표와 난이도 조정안을 생성한다.

## MVP 범위

1. 7x7 또는 8x8 보드 생성.
2. 같은 타입 타일 드래그 연결.
3. 3개 이상 연결 시 제거, 낙하, 리필.
4. 점수, 이동 횟수, 목표 카운트 UI.
5. 레벨 데이터 파일 또는 ScriptableObject.
6. AI 기획자 에이전트 초안:
   - 보드 분포 분석.
   - 평균 연결 가능 길이 추정.
   - 목표 점수, 이동 횟수, 타일 타입 수 제안.
   - 튜닝 결과를 Markdown 또는 JSON으로 저장.

## AI 기획자 에이전트 역할

에이전트 이름: Game Designer Agent

책임:

- 레벨 기획: 타일 타입 수, 목표 점수, 제한 이동 횟수, 장애물 비율 제안.
- 난이도 분석: 너무 쉬운 레벨, 막히는 레벨, 운 의존도가 높은 보드 탐지.
- 밸런싱 리포트: 플레이 로그를 읽고 "다음 수정안"을 사람이 이해 가능한 문장으로 정리.
- 포트폴리오 증거 생성: 매 튜닝 단계마다 `md/agent-reports/`에 리포트 저장.

초기 구현은 외부 API 없이 휴리스틱 에이전트로 시작한다. 이후 OpenAI API 또는 Agents SDK를 붙이면, 같은 인터페이스에 LLM 기반 에이전트를 추가한다.

## 개발 단계

### Phase 1: 플레이 가능한 퍼즐 코어

- 보드 생성, 타일 입력, 연결 검증, 제거/낙하/리필.
- Unity 에디터 메뉴에서 프로토타입 씬 자동 생성.
- 모바일 세로 화면 기준으로 카메라/보드 배치.

### Phase 2: 레벨/데이터 구조

- 레벨 목표, 이동 횟수, 타일 타입 수, 스폰 가중치 데이터화.
- 샘플 레벨 5개 작성.
- 플레이 로그 수집.

### Phase 3: AI 기획자 에이전트

- 보드/플레이 로그 분석기.
- 난이도 리포트 생성.
- 레벨 수정안 JSON/Markdown 출력.
- CLI에서 "분석 -> 제안 -> 적용" 루프 실행.

### Phase 4: 포트폴리오 마감

- 30~60초 플레이 영상.
- README에 기술 의도, AI 에이전트 구조, 튜닝 사례 정리.
- 지원서용 요약 문장 작성.

## 지원서용 한 줄

Unity 2D로 포코팡 스타일의 Line-Linker 퍼즐을 구현하고, 보드 상태와 플레이 로그를 분석해 레벨 난이도를 제안하는 AI 게임 기획자 에이전트를 설계했습니다.
